Voltar para Gestão

Sistema de Gestão

Esta aplicação foi desenvolvida com foco principal nas funcionalidades administrativas, como cadastro, controle, edição e gerenciamento de dados.

A interface pública (Home) não foi implementada intencionalmente, pois o objetivo do projeto é demonstrar as regras de negócio, a lógica do sistema e os processos internos de administração.

Portanto, a navegação deve ser realizada diretamente pelas telas administrativas.

Projeto acadêmico — foco em modelagem, lógica e operações do sistema.
